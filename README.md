# IT Moldova #

#### _IT Moldova_ is simple app that shows articles from ITMoldova.com web site. ####

## Features ##
 - View articles
 - Bookmark articles
 - Receive notifications
 - Choose between Light or Dark theme

## Built with ##
 - Kotlin
 - Room
 - Dagger
 - RxJava
 - Retrofit
